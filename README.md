# Software-Requirements-App-Development
Code for the final portion of our Software Requirements project.


To run the code on your machine follow these steps:

1. Make sure Java is installed on your machine.
2. Download the file UnifiedGamingPlatformGUI.java and store it wherever you want.
3. Go to the terminal and drill down into the folder you stored the java file in.
4. Run these two commands in the terminal:
   javac UnifiedGamingPlatformGUI.java
   java UnifiedGamingPlatformGUI.java
5. This Should launch the working application.
